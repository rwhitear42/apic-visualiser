globals

Global Modifiers

Global modifiers you add to your body tag to modify behavior or spacing globally.
Class Description
cui--animated Toggles global animation on across all patterns
cui--compressed Toggles compressed view on. Useful for data dense applications. Enables the 'compressed' spacing on all supported patterns.
cui--headerMargins Toggles h1-h6 header margins (margin bottom). By default, header elements have no margins.
cui--wide Toggles a wide container page bed (1600px). By default, the page bed is 1440px.
cui--sticky Toggles whether the header and footer are sticky (affixed to top and bottom of page). By default sticky is disabled.

Nudges

Use nudges to add extra margin (or padding) to your elements. Nudges are built off of the base spacing (20px) value so half-margin-left is equivalent to margin left 10px.
Margin Nudges
no-margin
no-margin-[top,right,bottom,left]
qtr-margin-[top,right,bottom,left]
half-margin-[top,right,bottom,left]
base-margin-[top,right,bottom,left]
dbl-margin-[top,right,bottom,left]

Padding Nudges

qtr-padding-[top,right,bottom,left]
