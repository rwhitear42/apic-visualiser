Downloads

There are several ways to access the Cisco UI kit:
1

Cisco CDN

You can now access the kit directly via Cisco's CDN (content delivery network). Simply paste the link below into the head element of your web page.

Basic
                    
<link href="http://www.cisco.com/web/fw/cisco-ui/1.3.4/dist/css/cui-basic.min.css" rel="stylesheet" type="text/css">
Standard
                    
<link href="http://www.cisco.com/web/fw/cisco-ui/1.3.4/dist/css/cui-standard.min.css" rel="stylesheet" type="text/css">
